PATTERN FOR REMAINING CLEANUP:
-------------------------------
All utility Makefiles follow the same pattern. They need removal of:

1. ifeq $(TARGET_OS),dos) ... endif blocks
2. ifeq $(TARGET_OS),aros) ... endif blocks  
3. ifeq $(TARGET_OS),morphos) ... endif blocks
4. ifeq $(TARGET_OS),amigaos) ... endif blocks

The amigaos block typically contains nested ifeq blocks for:
- BEBBO_TOOLCHAIN
- USE_CLIB2
- CRT_FLAGS selection
- DEBUG flags
- Compiler/linker flags (m68020-60, m68881)
- Include paths to oslibs/amigaos

EXAMPLE CLEANUP (from utils/qfiles/Makefile):
---------------------------------------------
BEFORE:
```
ifeq $(TARGET_OS),dos)
INCLUDES+= -I$(OSLIBS)/dos
LDFLAGS += $(OSLIBS)/dos/djtime/djtime.a -lc
endif
ifeq $(TARGET_OS),os2)
```

AFTER:
```
ifeq $(TARGET_OS),os2)
```

The same pattern applies to removing aros, morphos, and amigaos blocks,
preserving the os2, win32, win64, darwin, and unix blocks.
